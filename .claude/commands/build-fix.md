# ビルド修正

ビルドエラーを段階的に修正します：

1. PROJECT.md を参照してプロジェクトのビルドコマンドを確認・実行

2. エラー出力を解析：
   - ファイル別にグループ化
   - 重大度で並び替え

3. 各エラーについて：
   - エラーコンテキストを表示（前後 5 行）
   - 問題を説明
   - 修正案を提案
   - 修正を適用
   - ビルドを再度実行
   - エラーが解決されたか確認

4. 以下の場合に停止：
   - 修正で新しいエラーが発生
   - 同じエラーが 3 回の試行後も続く
   - ユーザーが一時停止をリクエスト

5. サマリーを表示：
   - 修正されたエラー
   - 残りのエラー
   - 新たに導入されたエラー

安全のため、一度に 1 つのエラーのみを修正してください！
